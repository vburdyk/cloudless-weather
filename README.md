# Terraform Lab 4

**Опис проєкту:** Лабораторна робота №4 (Варіант 1: Лічильник переглядів). Розгортання API з використанням безсерверних хмарних
сервісів, створення та використання модулів у Terraform.

**Передумови (Prerequisites):**
* Встановлені Terraform та AWS CLI.
* Налаштовані ключі доступу до AWS.
* Створений S3 бакет для збереження файлу стану (tfstate).

**Основні команди:**
* `terraform init` - ініціалізація робочої директорії та завантаження плагінів AWS.
* `terraform apply -auto-approve` - перевірка плану та створення ресурсів у хмарі.
* `terraform destroy -auto-approve` - повне видалення створеної інфраструктури.

**Тестування API:**
Після розгортання API можна протестувати за допомогою `curl`:
* Створити/збільшити перегляд: `curl -X POST <API_URL>/views/{page_id}`
* Отримати кількість переглядів: `curl -X GET <API_URL>/views/{page_id}`
